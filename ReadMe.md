# Unemployment Analysis During COVID-19

## Project Overview

This project analyzes unemployment trends in India before and during the COVID-19 pandemic using Python and data visualization techniques. The objective is to identify how unemployment rates changed across different regions and time periods.

## Dataset

The dataset contains information on:

* States and areas
* Unemployment rate (%)
* Employment figures
* Labour participation rate (%)
* Date and frequency of data collection

## Tools and Technologies

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Key Findings

* Unemployment increased significantly in 2020 compared to 2019 due to COVID-19.
* The highest unemployment spike occurred in April 2020 during the lockdown period.
* Tripura was the most affected region.
* Meghalaya was the least affected region.
* Urban areas experienced higher unemployment rates than rural areas.
* Unemployment rates declined after lockdown restrictions were eased.

## Conclusion

The analysis shows that the COVID-19 pandemic had a major impact on employment across India. Unemployment rates rose sharply during lockdown periods, particularly in urban areas, but gradually declined as economic activities resumed.
